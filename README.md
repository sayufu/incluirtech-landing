## :wrench:Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CityUp-UPC/citysos-landing.gitg
   ```
2. Install NPM packages in the cloned file
   ```sh
   npm install
   ```
3. Run the following command
   ```sh
   npm run dev
   ```
